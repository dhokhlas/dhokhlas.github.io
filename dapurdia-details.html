Dapurdia

dapurdia.netlify.app

Dapurdia adalah website katalog produk perlengkapan rumah tangga dan dapur yang dirancang untuk memudahkan pelanggan menemukan, melihat, dan menghubungi penjual secara langsung. Website ini memiliki tampilan modern, responsif, dan ramah pengguna sehingga dapat diakses dari berbagai perangkat.

Website ini dibuat menggunakan HTML, CSS, dan JavaScript untuk bagian front-end, dengan integrasi WhatsApp API agar pengunjung dapat langsung melakukan pemesanan atau konsultasi dengan penjual. Dapurdia menampilkan daftar produk dengan foto, deskripsi, dan harga secara jelas, serta mengutamakan kemudahan navigasi bagi pengguna.

Proyek ini dikembangkan untuk membantu bisnis lokal dalam meningkatkan visibilitas online dan mempermudah proses interaksi dengan calon pembeli melalui platform digital.

Fitur Utama

  • Desain responsif untuk semua perangkat
  • Integrasi WhatsApp untuk pemesanan cepat
  • Tampilan katalog produk yang rapi
  • Navigasi menu yang mudah digunakan
  • Informasi produk lengkap (gambar, deskripsi, harga)
  • Optimasi kecepatan loading halaman
